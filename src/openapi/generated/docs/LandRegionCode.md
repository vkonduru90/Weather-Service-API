# LandRegionCode

Land region code. These correspond to the six NWS regional headquarters: * AR: Alaska Region * CR: Central Region * ER: Eastern Region * PR: Pacific Region * SR: Southern Region * WR: Western Region 

## Enum

* `Ar` (value: `'AR'`)

* `Cr` (value: `'CR'`)

* `Er` (value: `'ER'`)

* `Pr` (value: `'PR'`)

* `Sr` (value: `'SR'`)

* `Wr` (value: `'WR'`)
